\[ \Delta \bigl(2^n - 1\bigr) = \sum_{k=0}^{n} 2^k - 1 \]

指を使った直感的理解

n ビットを指で表すと、指数関数的な増加が直感的にわかります：

指3本 → 2進数: 0111 → 十進数: 7
指数関数: 2³ = 8

指2本 → 2進数: 0011 → 十進数: 3
指数関数: 2² = 4

まとめると、2^n = 指で表した2進数の数値 + 1 という直感的対応が見えます。

コダックの定理 — 厳密解説

n ビットで表せる状態数は 2^n、 最大値は 2^n − 1 です。

状態数の導出

1ビットで取れる値は 0/1 の2通り。 2ビットなら 00, 01, 10, 11 の4通り。 一般化すると、n ビットで表現できる状態数は 2^n です。

最大値の考え方

全てのビットが1のパターンは、十進数で 2^n − 1。 例: 3ビット → 111 = 7 = 2³ − 1

応用・関連理論

情報理論

n ビットで表せる状態数 = 2^n は情報量の単位ビット(bit)の基本概念と直結します。

組合せ論

n 個の0/1選択肢がある場合の組み合わせ総数 = 2^n。 コダックの定理は、組合せ論的な視点でも理解できます。

コンピュータ実装

プログラミングでは n ビット整数型の最大値 = 2^n − 1。 オーバーフローの計算やビット演算を考えるときの基礎です。

まとめ

  • 状態数 = 2^n
  • 最大値 = 2^n − 1
  • 情報理論、組合せ論、コンピュータ表現の基礎原理に直結
  • 指での直感的理解 → Δ(2^n−1)=Σ(2^k−1) で数式的に表現可能
