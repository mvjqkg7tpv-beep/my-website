HTMLの基礎

HTMLはWebページの骨格です。要素（タグ）を組み合わせて構造を作ります。

  • <header>：ページのヘッダー
  • <main>：本文メイン部分
  • <footer>：ページのフッター
  • <section>：内容ごとの区切り
  • <article>：記事単位

CSSで見た目を制御

CSSで色・余白・フォント・レイアウトを調整できます。例：

body {
    background-color: white;
    color: #111;
    font-family: Arial, sans-serif;
}

さらに、ボタンやコードブロックの装飾も自由自在です。

JavaScriptで動きを追加

ユーザー操作に応じてページを動的に変えることができます。例：

document.querySelector('.btn').addEventListener('click', () => {
    alert('濃密な情報！');
});

実践的な知識まとめ

ここまでの内容を踏まえて、Web制作で意識すべきポイント：

  1. 構造（HTML）と見た目（CSS）を分離する
  2. 読みやすいフォント・余白・色使いを意識する
  3. 動的要素（JavaScript）は必要な箇所だけに使う
  4. 画像・図表で理解を補助する
  5. GitHub Pagesで公開すれば誰でもアクセス可能

応用例：GitHub Pagesでの公開

リポジトリにHTMLファイルを置き、Settings > Pagesで公開ブランチを指定するだけでOK。
